お店について

三重県津市大門ピッチャーズビルにあるお店です。建物はコンクリートのおしゃれな建物となっております。商品はアウトドアとエスニック系の洋服が置いてあるセレクトショップです。他にもアクセサリーや革製品やオリジナル商品などのアイテム展開があります。お店にchillくんという犬がいてとても可愛いです。

要望

初めの段階の要望としましては、３つありました。

  • ・エスニック感を出さずにシンプルなサイトがいい。
  • ・ブログをアップしたい。（出店やセール告知を行いたい）
  • ・オススメ商品欄が欲しい。

あとのデザイン等は全てお任せで良いとの事でした。

サイト制作にあたって

サイト制作にあたって、新たな事をするのでまずECプラットフォームを調べ始めました。次にどんな事ができるのか調べました。EC-CUBEを使用する理由は日本のプラットフォームというのも大きかったです。EC-CUBEではPHPフレームワークはSymfonyを使用するので、1年生の時にphpの学習はないのでまずはphpの基礎勉強とSymfonyの書き方の基礎を3週間程度学びました。その後EC-CUBEに触れ始めました。EC-CUBEのコミュニティ・勉強会に何度も参加しなんとか作り上げてきました。

サイト紹介

サイト紹介はPCのサイトを使用してサイトの紹介を行っていきます。

ファーストビュー

こちらはお店の方の意向でコンクリートのおしゃれな建物とお店の雰囲気をサイトに表示したいと要望がありこのような作りにしました。初めはただの横スライドで作成しましたが、ちょっと古いサイトに見えること高級感に欠けることから最終的に作り変えました。


下記はコードです。 スライドショーはswiperというライブラリを使用して作成しました。 swiperは参考サイトと公式を見ながら作成しました。 少し工夫した点はスライドが変わるたびにアニメーションをつけることです。スライドが変わるたびに画像がすこしずつアップされるアニメーションがついています。 


const myDelay = 7000;
let timer;

// アニメーション切り替え
const switchAnimation = () => {
  clearTimeout(timer);
  let activeSlide = document.querySelectorAll(
    ".mv02 .swiper-slide[class*=-active]"
  );
  for (let i = 0; i < activeSlide.length; i++) {
    activeSlide[i].classList.remove("anm-finished");
    activeSlide[i].classList.add("anm-started");
  }

  timer = setTimeout(() => {
    for (let i = 0; i < activeSlide.length; i++) {
      activeSlide[i].classList.remove("anm-started");
      activeSlide[i].classList.add("anm-finished");
    }
  }, myDelay - 2000);
};

// アニメーション終了
const finishAnimation = () => {
  let activeSlide = document.querySelectorAll(
    ".mv02 .swiper-slide.anm-started"
  );
  for (let i = 0; i < activeSlide.length; i++) {
    activeSlide[i].classList.remove("anm-started");
    activeSlide[i].classList.add("anm-finished");
  }
};

const mySwiper = new Swiper(".mv02 .swiper", {
  effect: "fade",
  fadeEffect: {
    crossFade: true,
  },
  loop: true,
  loopAdditionalSlides: 1,
  speed: 2000,
  autoplay: {
    delay: myDelay,
    disableOnInteraction: false,
    waitForTransition: false,
  },
  followFinger: false,
  pagination: {
    el: ".mv02 .swiper-pagination",
    clickable: true,
  },

  on: {
    slideChange: () => {
      finishAnimation();
    },
    slideChangeTransitionStart: () => {
      switchAnimation();
    },
  },
});

メニューバー

メニューは使いやすさ重視で考えました。 position:sticky,を使って追従するようにしました。
まずは左側の
”ショップロゴ”　トップページに戻れます。
”NEWS”　トップページのニュースの欄に飛びます。
”CATEGORY”　マウスカーソルを合わせると下にカテゴリーリストが表示されます。こちらはJSで作成しました。リストをクリックすると商品リストが検索された状態で表示されます。
“BRAND”　”CATEGORY”と同じような感じになっています。この辺りはEC-CUBEの変数を使用して作成しています。
“ABOUT”　トップページのお店の情報がのっている所へ飛びます。

携帯の時はハンバーガーメニューでカテゴリーとブランドを確認できます。
次にアイコンのところです。 左から”マイアカウント、お気に入り、検索、カートとなっています。 マイアカウントは別ページに飛びます。EC-CUBEの機能でアカウント登録が出来るので、その機能を活用しています。お気に入り商品を登録したり、住所等を登録でき購入時に手間を省くことができます。カートは商品をカートにいくつ入れているかと商品の合計の値段を確認できます。クリックするとカートページに飛ぶことができます。


検索はカテゴリ・ブランド・タグ・キーワード検索に対応しています。タグ検索については私が新たに機能をつけました。この機能をつけるうえで少し難しいところが多く参考サイトだけではなんともならないことも多くコミュニティとAIを駆使して作成しました。 セール・オススメ商品を検索が出来ることが重要だと考えたのでつけてよかったと思います。

NEWS（ブログ）

NEWSはブログページです。 EC-CUBEで機能を導入したくて、調べて色々してみましたが導入できませんでした。代案で浮かんだのがWordPressとmicroCMSです。microCMSは触った事がなく、WordPressはブログを作成する時に使用したので、投稿ページの使いやすさ等を知っているのもありWordPressで導入することにしました。サイトへの表示はrestAPIで表示させています。(phpで読み込んでtwigに変数で呼び出す）WordPressはテーマcocoonを作って簡易的に作成しました。

デザイン

私は自我が強いところがあってこのようにしたほうがいいと思う所は曲げなかったりします。そこは今後治すところだと思います。論理思考が故の合わないのがわかってるとかで意見を聞く機会を逃したりするのはあまりよくないです。参考サイトはいっぱい見ていい所はたくさん盗んで、実現していく。今回のサイトではそのようにしています。今後もこの習慣はつけていきたいと思います。また私はアクセントカラーに弱いように感じています。カラーを意識して作ることもデザインへの一歩だと思うのでもっと気を配り作るべきかなと思いました。

コーディングはhtmlは構想を気にしながら、css綺麗にBEMでまとめあげました。Javascriptは基本的に難しいことはしてないですが、わからないことが出てきた時はchatAIを通して確認をしていました。今回はphpとSymfonyがとても難しくそこに悩まされました。ただ、フロントからバックエンドの情報に触れたことで出来る事は見えるので楽しい言語であることを知ることもできました。いずれバックエンドの勉強も深く知りたいなと思います。まずはフロント面Javascriptと今後やっていきたいReactの学習を頑張りたいと思います。 あとECサイトはSEO,サイトスピードは命なのでこの方面でhtml,css,jsの学習をもっと深めるべきとも感じました。
